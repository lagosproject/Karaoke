# Security Policy

## Supported Versions

We actively support and patch the latest stable releases of SingChronized. 

| Version | Supported          |
| ------- | ------------------ |
| v2.x    | :white_check_mark: |
| v1.x    | :x:                |

## Reporting a Vulnerability

We take the security of SingChronized seriously. If you find a security vulnerability, please do **not** open a public issue. Doing so exposes the issue to the public before a fix can be prepared.

Instead, please report vulnerabilities responsibly by following these steps:

1. Send an email to **security@example.com** (replace with your secure email contact).
2. Include a detailed description of the vulnerability, steps to reproduce, and any proof of concept (PoC) code or screenshots.
3. Allow up to 48 hours for an acknowledgment of your report.

We will work to resolve the issue as quickly as possible and will keep you informed of our progress. Once a patch is released, we will credit you for the discovery (if desired).
